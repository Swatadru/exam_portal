## ✨ Features

### 👨‍🎓 Student Panel
- Secure Login & Registration  
- Attempt Exams (MCQ, MSQ, Numerical)  
- Real-Time Exam Timer with Alerts  
- Question Navigation Palette  
- View Detailed Results & Analytics  
- Reattempt Exams  

### 👨‍🏫 Admin Panel
- Create & Manage Exams  
- Add/Edit Questions  
- Manage Students  
- Monitor Results  
- Provide Explanations  

---

## 🧠 Exam Engine

- 📚 Multiple Sections (Physics, Chemistry, Mathematics)  
- 📌 Smart Question Status Tracking System  
- ⚖️ Custom Marking Scheme:
  - MCQ: +4 / -1  
  - MSQ: Partial Marking Supported  
  - Numerical: No Negative Marking  
- ⏱ Timer with Warning & Danger States  
- 📊 Accuracy & Score Calculation  

---

## 🛠️ Tech Stack

- React (CDN)  
- Babel (JSX Compilation)  
- KaTeX (Math Rendering)  
- LocalStorage (Data Persistence)  
- HTML5 + CSS3
